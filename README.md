# EZLynx Clone - React App

This is a **React-based EZLynx clone** that allows users to:

- Quote **Home and Auto insurance**
- Calculate premiums dynamically
- Add clients and policies
- Generate a downloadable **Declaration Page PDF**  

This project is built with:

- React 18
- React Router DOM for routing
- jsPDF for PDF generation
- Deployed on **Netlify**
- Hosted source code on **GitHub**
